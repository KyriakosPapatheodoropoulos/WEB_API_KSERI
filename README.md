# &nbsp;            # ADISE25\_2020133 - Παιχνίδι Ξερή (Web API)



### **Περιγραφή Project**



Το project αφορά την υλοποίηση του παραδοσιακού παιχνιδιού "Ξερή" σε μορφή Web API.

Η εφαρμογή ακολουθεί την αρχιτεκτονική Web API με backend σε PHP και βάση δεδομένων MySQL. 

Το παιχνίδι παίζεται από δύο παίκτες (Human vs Human) μέσω εντολών CLI (curl). 



### Βασικά Χαρακτηριστικά:

\- Πλήρης έλεγχος κανόνων παιχνιδιού (captures, πόντοι).

\- Αναγνώριση "Ξερής".

\- Διαχείριση σειράς παικτών και αναγνώριση τέλους παιχνιδιού (Deadlock).

\- Πλήρης αποθήκευση της κατάστασης του παιχνιδιού στη MySQL (Relational \& JSON).

\- Αυθεντικοποίηση χρηστών μέσω Tokens.





### **Αναλυτική Περιγραφή API**





###### 1)Αρχικοποίηση \& Σύνδεση

&nbsp;  **join\_game.php**: Επιτρέπει σε έναν παίκτη να εισέλθει στο παιχνίδι.

&nbsp;  Ο πρώτος παίκτης αρχικοποιεί το board (μοιράζει φύλλα κάτω και στον εαυτό του) και περιμένει αντίπαλο.

&nbsp;  Επιστρέφει ένα μοναδικό Token.



###### 2)Κατάσταση Παιχνιδιού

&nbsp;  **get\_game\_status.php**: Επιστρέφει σε μορφή JSON την τρέχουσα κατάσταση του παιχνιδιού (φύλλα στο τραπέζι, φύλλα στο χέρι του παίκτη, ποιανού σειρά είναι, score).



###### 3)Κινήσεις

&nbsp;  **make\_move.php**: Δέχεται το Token του παίκτη και την κάρτα που επιθυμεί να παίξει. 

&nbsp;  Ελέγχει αν είναι η σειρά του, αν η κάρτα είναι έγκυρη και εκτελεί τη λογική των μαζεμάτων (captures) ή της "Ξερής".



###### 4)Τέλος Παιχνιδιού \& Σκορ

&nbsp;  **get\_final\_score.php**: Υπολογίζει το τελικό σκορ βάσει των κανόνων (10 καρό, 2 σπαθί, Άσσοι, φιγούρες, πλειοψηφία φύλλων και Ξερές) και αναδεικνύει τον νικητή.



###### 5)Διαχείριση Παρτίδας

&nbsp; **create\_game.php**: Χρησιμοποιείται για την εσωτερική δημιουργία μιας νέας εγγραφής παιχνιδιού στη βάση δεδομένων, προετοιμάζοντας το έδαφος για την είσοδο των παικτών.

&nbsp; **abort\_game.php**: Επιτρέπει την ακύρωση ή τον τερματισμό μιας παρτίδας που βρίσκεται σε εξέλιξη, καθαρίζοντας την κατάσταση (status) στη βάση δεδομένων.



### 

### **Τεχνολογίες Ανάπτυξης**



**->Server-side**: PHP 8.x (Web API implementation)

**->Data Persistence**: MySQL Relational Database

**->Database Access**: PDO (PHP Data Objects) με χρήση Prepared Statements.

**->Data Exchange**: JSON format για την επικοινωνία μεταξύ API και Client.



##### **Οδηγίες Εγκατάστασης**



1\. Αντιγράψτε τα αρχεία στο `htdocs` του XAMPP.

2\. Εισάγετε το αρχείο `kseri\_db.sql` στη MySQL μέσω phpMyAdmin.

3\. Ρυθμίστε τα στοιχεία σύνδεσης στο `db.php`(Υπεύθυνο για τη σύνδεση με τη MySQL).
